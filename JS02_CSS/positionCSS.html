Position Static

position static digunakan untuk mengatur element menjadi static secara default. Elemen akan mengikuti posisi normal secara default, element tdk dipengaruhi oleh properti seperti top, bottom, left and right.

Elemen ini diatur menggunakan posisi hectic;




position: relative;

sebuah element yg menggunakan position relative akan terletak pada posisi normal. Mengatur properti atas, kanan, bawah, kiri dari elemen yg diposisikan dgn "relative" akan membuatnya jauh dari posisi normalnya. Konten lain tdk akan disesuaikan agar sesuai dgn celah yg ditinggalkan oleh elemen tersebut.:

Elemen ini diatur menggunakan posisi relative;




position: fixed;

sebuah element HTML yg di setting dgn position fixed akan memiliki sifat tetap, tanpa ada perubahan bahkan jika halaman website di scroll.:

Elemen ini diatur menggunakan posisi fixed;




















position: absolute;

Element HTML yg menggunakan position absolute akan diposisikan relatif dgn element lain yg mendahuluinya yg terdekat, bkn relatif terhadap layar secara normal.:

Elemen ini diatur menggunakan posisi relative;
Elemen ini diatur menggunakan posisi absolute;




position: sticky;

Cobalah utk scrollutk mengetahui bagaimana posisi sticky bekerja.

Catatan: IE/Edge 15 dan versi yg lebih lama tdk mendukung sticky.

Hola Aku adalah sticky!

Dalam contoh ini elemen sticky menempel pd posisi paling atas halaman(top:0), ketika mencapai posisi scrollnya.
